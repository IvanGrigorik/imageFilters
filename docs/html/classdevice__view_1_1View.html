Detailed Description

Device-side image view for CUDA kernels.

This lightweight class provides a device-side interface to image data. It stores only a pointer to device memory along with dimensions, avoiding any STL containers that cannot be used on the GPU. Used to pass image data to CUDA kernels efficiently.
